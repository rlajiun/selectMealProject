# 식당 예약 프로그램

## 소개
- Java로만 구현한 프로그램
- Java와 MVC 패턴, Sington 등에 대해 공부할 수 있었음
- 음식 메뉴 선택에 어려움이 있는 사람들에게 도움을 주기 위한 프로그램
- 음식 선택과 더불어 등록된 식당 예약을 할 수 있도록 구현
